# java-data-structures-projects

This repository contains simple menu-driven Java projects demonstrating the use of fundamental data structures.

## Projects

## 1. Railway Ticket Counter Simulation
Concepts used:
- Queue (customer line)
- Stack (ticket cancellation)
- Array (ticket storage)

Features:
- Add customer
- Serve customer
- Cancel ticket
- Display queue

## 2. Supermarket Billing Queue
Concepts used:
- Queue (customers waiting for billing)
- Stack (item reversal)
- Array (items and prices)

Features:
- Add customer
- Process billing
- Show pending customers
